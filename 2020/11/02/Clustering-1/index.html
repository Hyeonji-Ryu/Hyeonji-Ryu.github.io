작성일 2020-11-02 In Clustering

해당 시리즈는 라즈베리파이 3개를 연결하고 하둡(Hadoop)과 스파크(spark)를 사용하여 빅데이터를 다루는 과정을 보여줍니다.

라즈베리 파이 오픈

1.이미지 다운로드 (sdcard)

  1. lite 다운
  2. ssh 파일 만들기 -> touch ssh
  3. 랜선 연결
  4. ping raspberrypi.local
    ssh pi@raspberrypi.local

ssh-keygen -R raspberrypi.local

ssh pi@raspberrypi.local -> 초기 비번: raspberry

sudo apt-get update
sudo apt-get upgrade

sudo raspi-config

도커 설치

sudo curl -fsSL https://get.docker.com/ | sudo sh

sudo docker version

sudo usermod -aG docker pi

logout 후 재접속

마리아 디비 다운

docker pull linuxserver/mariadb

라즈베리파이 끄는 법

sudo shutdown -h now

파이썬 깔기

sudo apt-get install python3
