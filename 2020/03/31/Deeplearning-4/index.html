4. 경사하강법

작성일 2020-03-31 Edited on 2020-04-02 In Deep learning with Julia

해당 시리즈는 프로그래밍 언어 중 하나인 줄리아(Julia)로 딥러닝(Deep learning)을 구현하면서 원리를 설명합니다.

이전 장에서 우리는 손실 함수에 대해서 살펴보았다. 손실 함수는 신경망을 평가하는 판단지표로서 사용되며, 손실 함수의 값을 줄이는 방향으로 신경망을 학습시킨다고 하였다. 다른 말로, 좋은 신경망은 손실 함수 값이 작아야 한다. 그렇다면 이쯤에서 “어떻게 손실 함수의 값을 줄일 수 있을까?”라는 의문이 들 것이다. 이것이 바로 최적화의 문제이다. 이번 글에서는 손실 함수를 최소화하는 경사하강법에 대해서 살펴볼 것이다.

경사하강법이란

경사하강법은 손실함수의 값을 최소화하는 가중치와 편향을 찾는 최적화 알고리즘이다. 경사하강법의 작동 과정을 순서대로 나열하면 다음과 같다.

  1. 임의의 값을 가진 가중치와 편향을 생성한다.

  2. 신경망 계산을 한다.

  3. 손실 함수로 출력층의 결과인 예측값과 정답을 비교한다.

  4. 예측값과 정답의 격차를 줄이기 위해 가중치와 편향을 조정한다.

  5. 2~4의 과정을 반복하여 손실 함수의 값을 줄인다.

위의 순서는 경사하강법의 프로세스를 전체적으로 설명한 것이다. 위의 모든 과정들이 중요하지만, 우리가 이번 장에서 집중적으로 다룰 부분은 가중치와 편향 조정 방법이다.

수학적 정의

경사하강법에서 가중치와 편향은 어떤 기준으로 조정되는가? 이를 알기 위해서는 손실 함수와 가중치와의 관계를 수학적으로 이해해야 한다. 먼저 손실 함수를 살펴보자. 아래의 수식은 손실 함수 중 ‘평균 제곱 오차’이다.

평균 제곱 오차는 예측값과 정답을 뺀 오차에 제곱하여 평균을 구한다. 이 수식에서 평균은 일반화를 위한 기술적 장치이기 때문에 제외하고 제곱 오차만 살펴보자. 제곱 오차의 수식은 다음과 같다.

여기서 예측 값 $y_i$는 층의 출력 값인 $XW+B$이기 때문에 이로 대체한다.

$((XW+B)-t_i)^2 \
= (XW+B)^2-2t_i(XW+B)+t_i^2 \
= XW^2+(2B-2t_i)XW-2t_iB+B^2+t_i^2$

위 식에서 $X, T$ 값은 이미 정해져 있기 때문에 상수이며, 가변적인 $W, B$를 기준으로 한다. $W^2$의 계수는 $X^2$이기 때문에 양수이며, 따라서 이를 그래프로 그리면 아래와 같은 3차원 2차 함수의 형태를 가진다. 아래의 그래프는 축의 숫자를 신경쓰지 말고 모양만 참고하자.
